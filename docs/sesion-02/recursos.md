# Recursos Adicionales

## Documentación Oficial

### Testing Fundamentals

- **[Testing Library - Official Docs](https://testing-library.com/docs/)** - La guía definitiva para testing de interfaces de usuario. Incluye React, Vue, Angular y más.
  - [Queries](https://testing-library.com/docs/queries/about) - Cómo buscar elementos en el DOM
  - [User Interactions](https://testing-library.com/docs/user-event/intro) - Simular acciones de usuario
  - [Async Utilities](https://testing-library.com/docs/dom-testing-library/api-async) - waitFor, findBy, etc.

- **[Jest Documentation](https://jestjs.io/docs/getting-started)** - Framework de testing que usamos
  - [Expect API](https://jestjs.io/docs/expect) - Todas las assertions disponibles
  - [Mock Functions](https://jestjs.io/docs/mock-functions) - Cómo mockear dependencias
  - [Asynchronous Testing](https://jestjs.io/docs/asynchronous) - Testing de código async

### Backend Testing

- **[Supertest - GitHub](https://github.com/visionmedia/supertest)** - Testing de APIs HTTP
  - Simple y potente para testear Express/Node.js
  - Ejemplos de uso con diferentes frameworks

- **[Mock Service Worker](https://mswjs.io/docs/)** - Interceptor de requests HTTP
  - [Getting Started](https://mswjs.io/docs/getting-started) - Setup inicial
  - [Request Handlers](https://mswjs.io/docs/basics/request-handler) - Definir mocks
  - [Browser Integration](https://mswjs.io/docs/integrations/browser) - Usar en development

### React Testing

- **[React Testing Library](https://testing-library.com/docs/react-testing-library/intro)** - Testing específico de React
  - [Example](https://testing-library.com/docs/react-testing-library/example-intro) - Ejemplos prácticos
  - [Cheatsheet](https://testing-library.com/docs/react-testing-library/cheatsheet) - Referencia rápida

- **[React Hooks Testing Library](https://react-hooks-testing-library.com/)** - Testing de custom hooks
  - [renderHook API](https://react-hooks-testing-library.com/reference/api) - Documentación completa
  - [Examples](https://react-hooks-testing-library.com/usage/basic-hooks) - Casos de uso comunes
